# Security Policy

## Supported Versions

This research project is maintained from the `main` branch.

## Reporting a Vulnerability

If you find a security issue, please open a private report through GitHub's
security advisory flow if it is enabled for the repository. If private reporting
is not available, open a minimal public issue that avoids sensitive exploit
details and ask the maintainer for a private contact path.

Please do not publish working exploits or private credentials in issues, pull
requests, or discussions.
